Guía rápida

Todo lo que necesitas para dejar el sistema funcionando.

1. Conecta Firebase. Abre el archivo js/config.js y pega los datos de tu proyecto de Firebase (Firestore activado).

2. Publícalo. Sube esta carpeta a GitHub Pages, Netlify o cualquier hosting con HTTPS — la cámara solo funciona en HTTPS (o en localhost).

3. Registra tu catálogo. Ve a la pestaña "Catálogo" y agrega tus productos con su código EAN.

4. Vende. Ve a "Vender", inicia el escáner y apunta al código de barras. Si un producto no tiene código legible, búscalo manualmente.
